ОбзорПроцесс

Цель проекта — создать айдентику для сайта с развивающими играми и дизайн главной страницы с учётом поисковой оптимизации. Также интерфейсные элементы нужно анимировать.

Сначала пишу основные идеи без форматирования. Они будут призывать пользователя совершать целевое действие.

Накидываю юзерфлоу и рисую вайрфреймы для основных страниц, чтобы главная страница гармонично сочеталась с остальными. Продумываю общие структурные решения.

В дальнейшем буду работать только с главной страницей.

Разрабатываю стиль и логотип.

Логотип

Иллюстрации взяты с фотостока, в Блендере моделирую и анимирую лампочку.

На основе выписанных идей формулирую заголовки и сопровождающий текст для каждого модуля. Проверяю написанный текст на тошноту, исправляю недостатки. Готовый текст добавляю в макет.

В анимации продумываю эффекты «открывающихся» модулей, прозрачности текста, интерлиньяжа.
